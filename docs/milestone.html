Accomplishments and Preliminary Results

video | slides

Clouds

We were able to edit Sebastian Lague's code and generate a smooth transition of clouds between stormy and sunny days (see video).

We first tested and found the cloud parameters (scale and density) that replicates clouds in stormy weather and sunny weather best. Then, we used Unity's Update function to implement a smooth transition between them using linear interpolation.

Lightning

As for lightning simulation, we decided to follow this paper after researching on previous works.

We used LightningBranch script to start from a main "trunk", and then recursively generate the branches. Then we used LightningSegment script to create a cylinder with a purple material to simulate a small segment of the lightning given the starting point, length and direction. We are looking to replace this step with shader rendering in the future.

Progress Check & Next Steps

We didn't really follow our initial plan that would allow us to reach our aspirational deliverables. But, we are still on track for completing our baseline plan, which is to complete lightning simulation.

Our plan is to generate a good geometric representation for lightning bolts this week. Then, we will focus on the aesthetics which will include adding a glow effect, variating the luminance of lightning streaks based on its location, as well as cloud dependent spawning of our lightning bolts.
